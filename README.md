# Wochentags Rechner

Hier findest du den Wochentags Rechner als fertige jar-Datei, oder als Netbeans Projekt.

## Benutzung
1. Wähle den Tag
2. Wähle den Monat
3. Gib das Jahr an (in XXXX Format)
4. Drücke auf "Rechne"

Und schon hast du dein Ergebnis

(Schaltjahre funktionieren noch nicht!)
